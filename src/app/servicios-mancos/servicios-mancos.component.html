creacion de aplicacion de gestion de escritorio

  • creacion de aplicaciones con java y c# lanzadas sobre bases de datos SqlServer o mySql


    información adicional

    Funcionalidad:

    gestiona tu empresa comprando tu aplicacion personalizada que te permitirá llevar la gestion de toda la informacion de la empresa en un entorno amigable
    Las funcionalidades principales son:

    • Gestion de Usuarios o clientes
    • Gestion de empleados
    • Productos y proveedores
    • TPV

    Precio:

    El precio varia en funcion de el numero de horas que se dedique al proyecto estas estarán condicionadas al numero de componentes que se van a gestionar
    Otro a tener encuenta sera sobre que base de datos sera lanzada la aplicacion, si es sobre SqlServer tendra un mayor costo que si es sobre MySql
